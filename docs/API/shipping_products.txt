Use case: Nhà máy sản xuất gửi lô sản phẩm đến đại lý

Điều kiện: Token nhà máy sản xuất

Post: /api/v1/products/shipping

Request:
{
    "to": id của đại lý phân phối,
    "lot_number": số lô vận chuyển
}

Response:
Thành công:
200 
{
    "data": "shipping id"
}

Request thiếu trường:
400
{
    "err": "Fill empty field"
}

Người gửi và người nhận không đúng role (hiện tại chỉ cho phép production gửi cho distribution):
400
{
    "err": "Can not send"
}

Lỗi khác:
500
{
    "err": message error
}
